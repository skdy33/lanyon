## what is spring cloud
* open source library다
* **Cloud Connector**: cloud 서비스를 만들기 쉽게 하기위해 만들어졌다.
	* 이걸 사용하면 application이 특정 서비스를 발견하고 연결하기가 쉽다. 원래는 쓸데없는 configuration 같은걸 해야 하잖아?
